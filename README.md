This repository provides the implementation of the methods and experiments presented in:

Hai-Feng Zhang, Kun-Peng Liu, Huan Wang, and Chuang Ma, “Reconstructing network structures using Gaussian mixture model: From unsigned to signed networks,” *IEEE Transactions on Systems, Man, and Cybernetics: Systems* **55**(11), 8544–8557 (2025). doi:10.1109/TSMC.2025.3606222.

For questions regarding the code, usage, or the paper, please contact Chuang Ma at [chuang_m@126.com](mailto:chuang_m@126.com).
